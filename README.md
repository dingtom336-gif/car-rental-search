# car-rental-search

Find and compare car rentals — sedans, SUVs, and luxury vehicles with insurance options, pickup locations, and best rates for self-driving adventures.

## Overview

Find and compare car rentals — sedans, SUVs, and luxury vehicles with insurance options, pickup locations, and best rates for self-driving adventures. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install car-rental-search
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/car-rental-search
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Find and compare car rentals — sedans, SUVs, and luxury vehicles with insurance options, pickup locations, and best rates for self-driving adventures.
